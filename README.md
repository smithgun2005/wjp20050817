# 学生个人学术主页

这是一个静态个人学术网页模板，直接打开 `index.html` 就能预览。

## 常用修改位置

- `index.html`：修改姓名、学校、研究方向、自我介绍、论文、项目经历和联系方式。
- `styles.css`：修改颜色、字号、间距和响应式布局。
- `script.js`：控制论文筛选、移动端导航和复制邮箱提示。
- `assets/academic-workspace.png`：首页横幅图，可以替换成你的实验室、校园、研究图或个人工作照。

## 上线建议

- 把 `your.name@example.com` 替换成真实邮箱。
- 把 `#` 链接替换为 PDF、GitHub、Google Scholar、CV 等真实地址。
- 如果使用 GitHub Pages，保持这些文件在仓库根目录即可。

## GitHub
